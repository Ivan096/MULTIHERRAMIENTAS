# MULTIHERRAMIENTAS

# DESCRIPCIÓN:
La aplicación esta desarrollada en lenguaje java; la cual trabaja con una base de datos, esta base de datos debe estar alojada dentro del equipo computacional donde se instalará el programa. El propósito de esta aplicación es la automatización de tareas relacionadas con el control de inventarios en la empresa de “MULTIHERRAMIENTAS”

# PROBLEMA IDENTIFICADO:
Las personas de la empresa no cuentan con un sistema para la administración del inventario por lo que ellos contabilizan manualmente mediante hojas de Excel las cuales no siempre se encuentran con información actualizada por lo que al final del día tienen que volver a contabilizar todo el inventario.

# SOLUCIÓN:
Se desarrollo un software el cual ayudará con la gestión del inventario ya que solo la contabilización del producto lo hará automáticamente, las personas de la empresa solo tendrán que ir poniendo las salidas o entradas del producto; además de ello se deja abierta la posibilidad de integrar mas funciones al software en un futuro.
